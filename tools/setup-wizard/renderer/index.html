ようこそ！

このウィザードは、ガジェット開発を始めるための準備を、順番に案内します。

  • ⏱ 所要時間: 約 15 分
  • 🧰 やること: 環境チェック → GitHub 連携 → リポジトリの取得 → 自分のガジェットの雛形作成
  • 💡 途中で失敗しても大丈夫。各画面に「手動で続行する方法」があります

環境診断

開発に必要なツールがインストールされているか確認します。

診断中…

✗ がある場合: ダウンロードボタンから公式インストーラーを入手してインストールし、 「再診断」を押してください（インストール後に再診断しても ✗ のままの場合は、 このアプリを一度閉じて開き直してください）。

手動で確認する場合

ターミナル（コマンドプロンプト）で次を実行し、バージョンが表示されればOKです:

git --version
node --version

手動で続行する場合

認証がうまくいかない場合も、GitHub アカウントさえあれば手作業で進められます。 ブラウザで github.com にログインした状態で、次の手順（手順4の手動版）へ進んでください。

ターミナルでリポジトリのフォルダに入り:
