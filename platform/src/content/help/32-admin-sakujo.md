# 入居者アカウントの削除

誤って登録されたアカウントや、使われなくなった空きアカウントを、大家が片づけるための
機能です。**削除は元に戻せません。** 事故を防ぐため、対象の確認を打ち込む多段の確認を
挟んでから実行します。

## 削除するとどうなるか

- そのアカウントの**個人データ（プロフィール・設定・保存データ・連携キー）はすべて消えます**。
  **元に戻せません。**
- その人が**作った道具は、既定では長屋に残ります**（作り手が外れて「大家預かり」になります）。
  道具そのものや作者の権利は消えません。

## 手順

### 手順1 対象を選ぶ

大家の間（管理画面）の「入居者」一覧で、消したい人の行の「**削除**」を押します。
確認のダイアログが開きます。

### 手順2 確認の打ち込み

- ダイアログに、対象の**メールアドレス**（メールが無い人は表示名）が示されます。
- そのとおりに**正確に打ち込む**まで、「削除する」ボタンは押せません。
  別人を誤って消さないための仕組みです。
- 打ち込みが一致したら「削除する」を押して実行します。

## 消せない相手（安全の歯止め）

次の場合は削除できません。

- **自分自身は削除できません。** 自分のアカウントを消したいときは、退去（後述）を使います。
- **最後の大家は削除できません。** 大家がひとりもいなくなり、誰も管理できなくなるのを
  防ぐためです（「最後の大家は削除できません」と出ます）。先に別の入居者を大家にしてください。

すべての削除は記録（監査ログ）に残ります。

## 「大家による削除」と「本人による退去」の違い

似た結果になりますが、入り口が違います。

| | 大家による削除 | 本人による退去 |
|---|---|---|
| 誰が行う | 大家（他人のアカウントを削除） | 本人（自分のアカウントを削除） |
| 主な用途 | 誤登録・空きアカウントの片づけ | 自分から長屋を出る |
| 消えるもの | 対象の個人データ（元に戻せない） | 自分の個人データ（元に戻せない） |
| 最後の大家 | 削除不可 | 退去不可（先に大家を指名） |

どちらも作った道具は既定で長屋に残り、作者の権利は移りません。

## 関連する案内

- 管理画面全体のきほん → 案内所の「大家の間（管理のきほん）」
- 作った道具を道具市から下げたいとき → 案内所の「道具の緊急停止」
