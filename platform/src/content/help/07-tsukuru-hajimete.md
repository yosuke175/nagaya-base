# はじめての道具づくり（GitHubアカウントから）

道具（ガジェット）を作るのは、**PCへのインストールは不要**で、クラウド上でAIと会話しながら進められます。
ただし最初だけ「GitHub（ギットハブ）」という**コードの置き場所**の準備が要ります。ここではその一番最初、
**GitHubアカウントを持っていない状態から**、順番に説明します。あわてず一つずつで大丈夫です。

> ※ 各画面の写真（スクリーンショット）は順次追加していきます。まずは文章と下の図で全体の流れをつかんでください。

## 全体の流れ（この図がゴールの地図です）

![道具づくりの流れ](/img/guide/tsukuru-flow.svg)

要するに「**①GitHubに登録 → ②長屋を自分のGitHubにコピー → ③クラウドで作る → ④pushする → ⑤部屋で試運転 → ⑥申請して公開**」です。

## そもそも GitHub とは？

GitHub は、プログラムのコードを保管・共有する**世界的な倉庫サービス**です（無料で使えます）。
NAGAYA-BASE のコードもここに置いてあります。あなたは、この倉庫に**自分用のコピー**を作って、そこで道具を育て、
できたら「本家に入れてください」と申請します。

## ⚠ 大家（管理人）の方へ

あなたが**本家リポジトリの持ち主（大家）**の場合は、**手順2の「コピー（Fork）」は不要**です
（自分のものは Fork できません）。本家で直接ブランチを作って作業してください。
2号室などの**一般の入居者は、下の手順どおり Fork します**。

## 手順1　GitHubアカウントを作る（持っていない場合）

1. ブラウザで [github.com/signup](https://github.com/signup) を開く
2. **メールアドレス**を入力して進む
3. **パスワード**、**ユーザー名**（半角英数字。これがあなたのIDになります）を決める
4. GitHubから**確認コードのメール**が届くので、その番号を入力する
5. 「無料（Free）」プランを選ぶ

これでアカウントは完成です。**ユーザー名は後で使う**ので覚えておいてください。

## 手順2　長屋を自分のGitHubにコピーする（Fork）

「Fork（フォーク）」＝**本家の倉庫を、まるごと自分のGitHubに複製する**ことです。

1. GitHubにログインした状態で、[本家を Fork する](https://github.com/yosuke175/nagaya-base/fork) を開く
2. 「Create fork」を押す
3. 少し待つと、**あなたのGitHub**（`github.com/あなたのユーザー名/nagaya-base`）に複製ができます

これで、あなた専用の作業コピーができました。ここに道具を作っていきます。

## 手順3　クラウドの作業場で開く（PCに何も入れません）

コードを**書いて動かす場所**が必要です。自分のPCに落とさず、クラウドの作業場で開きます。どちらか一つでOK:

- **GitHub Codespaces**: 自分のForkのページで「Code」→「Codespaces」→「Create codespace」。ブラウザの中にコードを書く画面が開きます。
- **Claude Code などのAIツール**: 自分のFork（`あなたのユーザー名/nagaya-base`）を開くように指示します。

## 手順4　AIと一緒に作って、GitHubに push する

1. AIに、`docs/gadget-spec.md`（道具の説明書）と `gadgets/_template`（雛形）を読ませる
2. 「こういう道具が作りたい」と伝える（**指示文は長屋の「工房 → 新しい道具をつくる」**に用意してあります。コピーして貼るだけ）
3. できたら **GitHubに保存（push / コミット）**する（Codespacesやツールのボタンでできます）

**push した内容が、次の試運転に反映されます。**

## 手順5　自分の部屋で試運転して確認する

1. 長屋にログインし、メニュー「**工房**」を開く
2. 「**部屋で試運転**」に、**あなたのGitHubユーザー名／ブランチ（ふつうは main）／道具のID**を入れる
3. 「試運転する」を押すと、作りかけが**あなたの部屋の中で本物として動きます**（まだ公開はされません）

うまく動かない所は、AIに「ここが動かない」と伝えれば直してくれます。直したら**またpush → もう一度試運転**。

## 手順6　できたら申請して公開する

1. 納得できたら、GitHubで「**Pull Request（プルリク）**」を作る＝「この変更を本家に入れてください」という申請
2. 取り込まれ（マージされ）たら、工房で「**道具市に公開する**」を押す
3. これで道具市に並び、みんながインストールできます

## 困ったとき

途中でつまずいても大丈夫です。**管理人・ベテラン職人**に、どの手順のどこで止まったかを伝えて聞いてください。
「案内AI」にも質問できます（AI設定済みの場合）。
