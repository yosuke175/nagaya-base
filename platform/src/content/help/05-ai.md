# AIの使い方・APIキーの取り方

## なぜ自分のAPIキーが要るの？

AIを使う道具（ガジェット）は、文章の生成などに「AIサービス」を使います。
その利用料は使った人が負担するのが公平なので、**あなた自身のAPIキー**を登録して使います
（BYOK＝Bring Your Own Key。使った分だけ、あなたが契約しているAIサービスに課金されます。
目安は1回の相談で数円ほど）。

キーは NAGAYA-BASE のサーバーで暗号化して保管され、**道具（ガジェット）には渡りません**。
AIの呼び出しもサーバー側で代行するので、キーがブラウザに出ることもありません。

## どのAIが使えるの？

お好みの提供元を選べます。今は次の3つに対応しています（右上「AI設定」で選択）:

| 提供元 | 例 | キーの形 |
|---|---|---|
| Claude（Anthropic） | claude-haiku など | `sk-ant-…` |
| OpenAI（ChatGPT） | gpt-4o-mini など | `sk-…` |
| Google（Gemini） | gemini-2.0-flash など | `AIza…` |

料金・速度・使い慣れで選んでください。他社の対応も順次ふやします。

## キーの取り方（各社）

### Claude（Anthropic）
1. [console.anthropic.com](https://console.anthropic.com/) にログイン（アカウントがなければ作成）
2. 左メニュー **API Keys** → **Create Key**
3. 表示された `sk-ant-…` をコピー（**この画面でしか全体を見られません**）
4. 支払い方法の登録が必要な場合があります（Billing）

### OpenAI（ChatGPT）
1. [platform.openai.com](https://platform.openai.com/api-keys) にログイン
2. **API keys** → **Create new secret key**
3. 表示された `sk-…` をコピー
4. **Billing** でクレジット（前払い）または支払い方法の登録が必要です

### Google（Gemini）
1. [Google AI Studio](https://aistudio.google.com/app/apikey) にログイン
2. **Create API key** を押す
3. 表示された `AIza…` をコピー
4. 無料枠があります（上限あり）。多く使う場合は請求先の設定を

## NAGAYA-BASE に設定する

1. 画面右上の **「AI設定」** を開く
2. **AIの提供元**を選ぶ（Claude / OpenAI / Google）
3. コピーした**APIキー**を貼り付ける
4. モデルは既定のままでOK（変えたい場合だけ書き換え）
5. **保存**

これで、`ai` を使う道具（例: スケジュール秘書の「AIに相談」）が動きます。

## 注意

- APIキーは**パスワードと同じ**です。人に教えたり、チャットに貼ったりしないでください
- キーが漏れたと思ったら、各社の画面でそのキーを削除（無効化）して作り直してください
- 使いすぎが不安なら、各社の画面で**利用上限（予算）**を設定できます
