# はじめての方へ

ここは、勉強会のメンバーが自作の「道具（ガジェット）」を持ち寄って組み合わせる長屋です。
入り方は2つ。どちらもいつでも変えられます（メニューの「案内」からやり直せます）。

## 店子（たなこ）— つかう人

1. 「道具市」をのぞく
2. 気になる道具を「インストール」する（必要な権限を確認して承認）
3. 自分の「棚」に並ぶ。道具はいつでも足したり外したりできます

欲しい道具が見つからないときは、「欲しい道具を提案する」から職人に伝えてください。
あなたの「欲しい」が職人の次の仕事になります。

## 職人（しょくにん）— つくる人

1. メニューの「案内」→「職人のはじめ方」からセットアップウィザードを入手
2. ウィザードの「次へ」に従うと、約15分で最初の道具の雛形ができます
3. 雛形を AI に読ませて改造 → PR を出すと道具市に並びます

プログラミング経験は不問です。AI に全部書かせて構いません。

## 困ったら

- 仕様の疑問: GitHub の Issue へ
- 勉強会メンバー: 勉強会のチャンネルでも OK
