# 大家の間（管理のきほん）

大家（管理者・admin）だけが入れる管理画面が「大家の間」です。ここで入居者の一覧を見て、
それぞれの立場（ロール）を切り替えられます。**自分自身の立場だけは、ここでは変えられません**
（うっかり大家でなくなって締め出される事故を防ぐためです）。

## 大家の間でできること

- **入居者の一覧を見る** — 表示名・メール・号室・いまの立場が並びます
- **立場（ロール）を付ける／変える** — 軒先（guest）・入居者（user）・大家（admin）から選ぶ
- 号室（部屋番号）の割り当て
- 道具の緊急停止・再開（案内所の「道具の緊急停止」を参照）
- 入居者アカウントの削除（案内所の「入居者アカウントの削除」を参照）
- 今月のAI利用のおおよその集計

このうち**立場を変える操作は、変更の記録（監査ログ）に必ず残ります**。誰が・いつ・誰を
どう変えたかが残るので、あとから確認できます。

## 立場（ロール）とは

長屋には3つの立場があります。

| 立場（表示） | 内部の呼び方 | できること |
|---|---|---|
| 軒先（ゲスト） | guest | 道具市を見るだけ。道具は使えません |
| 入居者 | user | 道具を使う・設定する、道具を作って公開する |
| 大家（管理者） | admin | 上に加えて、この「大家の間」の管理すべて |

- メールで登録して入居した人は、はじめから**入居者（user）**になります（自動）。
- **大家（admin）に上げられるのは、大家だけ**です。信頼できる人にのみ渡してください。
- 「職人」「店子」は入居者（user）の中の**自己申告の呼び方**で、立場としての違いはありません。

### 手順1 立場を変える

- 大家の間の「入居者」一覧で、対象の人の行にある選択肢から新しい立場を選びます。
- 選んだ瞬間に反映され、画面上部に「〇〇さんを△△にしました」と出ます。
- 記録（監査ログ）に自動で残ります。

### 注意 自分の立場は変えられない

自分自身の行で立場を変えようとすると、はじかれます
（「自分自身のロールはここでは変更できません」）。これは、最後の大家が誤って自分を
降格させ、誰も管理できなくなるのを防ぐための仕組みです。

## なぜ「見た目で隠す」だけでは足りないのか

大家の間は大家にしか表示されません。しかし NAGAYA-BASE では、**画面で隠すだけを
安全とはみなしません**。立場の変更や管理の操作は、

- **サーバー側**でも「本当にその人が大家か」を確かめ、
- **データベース側の番人（RLS）**でも、正しい立場の人以外は書き込めないようにする、

という**二重の守り**になっています。だから、仮に画面を細工しても、大家でない人が
勝手に立場を書き換えることはできません。安心して運用してください。
