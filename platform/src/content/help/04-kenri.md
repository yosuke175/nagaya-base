# 権利について

道具やコードを作る・使うときの権利の早見表です。むずかしく考えず、まずは下の表を。

## まず読む早見表（行為ごと）

| やりたいこと | 無償・勉強会内 | 自分の商売（有償） |
|---|---|---|
| 長屋の道具を使う・学ぶ | ○ 自由 | ○ 社内利用も自由 |
| 自分が書いたコードを転用する | ○ 自由 | ○ 自由（あなたの著作物） |
| 他人のコードを読んで学び、ゼロから書く | ○ 自由 | ○ 自由（アイデアは著作権の外） |
| 他人のコードをコピーして製品の主要部分に | ○ 自由 | △ 相談（原作者＋管理人の同意） |
| プラットフォームごと複製して販売 | — | ✕ 不可（協会の商用権） |

AIに書き直させても、コピーはコピーです（著作権は「表現」を守るもので、
書き直しの手数では消えません）。売り物には他人のコードを混ぜず、
学んで自分で書くのが基本です。

## 権利の要点

| 項目 | 内容 |
|---|---|
| あなたのコードの著作権 | **あなたに帰属したまま**（譲渡ではない） |
| 自分の貢献の外部利用 | 自由（本プロジェクトの外でも使える） |
| プロジェクト側の利用 | 商用を含む利用許諾（BSL 1.1 / 4年後に Apache 2.0 へ） |
| 収益の分配 | なし。クレジット表記で貢献を記録 |
| あなたの責任 | 第三者の権利を侵害しないこと（**AI生成コードも同様**） |

## 商用利用の相談

他人のコードを有償の製品に組み込みたいときは、事前に相談してください
（①原作者の同意 → ②管理人の許諾）。**自分のコードだけを使う場合、相談は不要**です。
まずは管理人（大家）に声をかけてください。
