# よくある質問

## お金はかかりますか？

かかりません。プラットフォームは無料枠で運営されています。
ただし AI を使う道具は、**あなた自身の AI API キー**の利用量を消費します
（使った分だけ、あなたの Anthropic アカウントに課金されます。目安: 1回の相談で数円）。

## CLA（貢献者許諾契約）って何ですか？

道具やコードを提出する人に同意してもらう約束事です。要点:

- **あなたの書いたコードの著作権はあなたのまま**（譲渡ではありません）
- プロジェクト側は商用を含めて利用できる（将来の事業化に備えるため）
- 収益の分配はなし。貢献はクレジット表記で記録

初回の PR 時に bot が確認します。詳細は「権利のはやわかり」へ。

## 自分のデータは安全ですか？

- 道具のデータは「あなた×その道具」専用の領域に隔離され、他の人・他の道具からは見えません
- API キー等はサーバー側で暗号化して保管され、道具には渡りません
- 道具は sandbox の中で動き、宣言外の通信はブロックされます

## 公開した道具に問題が見つかったら？

大家（管理者）が即時停止できます。停止理由は作者に通知されます。

## スマホでも使えますか？

使えます（PWA としてホーム画面に追加できます）。道具づくりは PC が必要です。
