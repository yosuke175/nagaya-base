# クラウドで道具を作る（Codespaces / Claude Code）

**要約**: 道具（ガジェット）づくりは、自分のPCに何もインストールせず、**クラウドの作業場**の中でAIと会話しながら進められます。自分のGitHubコピー（Fork）をクラウドの作業場で開き、AIに説明書と雛形を読ませ、工房で作った**指示文**を貼るだけです。PCの環境（セットアップウィザード）を使う方法もありますが、それは任意の選択肢です。

はじめての方は、先に案内所の「はじめての道具づくり（手順）」でForkまで済ませておいてください。この記事は、そのForkを「どこで開いて、どう作るか」を掘り下げます。

## クラウドの作業場とは

「作業場」とは、コードを書いて動かす場所のことです。ふつうは自分のPCに開発環境を入れますが、NAGAYA-BASE では**ブラウザやAIツールの中だけで完結**できます。自分のPCを汚しません。

選択肢は次のとおりです。どれか一つでOKです。

- **GitHub Codespaces** — ブラウザの中にコードを書く画面が丸ごと開く（GitHub純正）
- **Claude Code などのAIツール** — AIに自分のForkを開くよう指示して、会話で作る
- **（任意）自分のPC** — セットアップウィザードで環境を自動でそろえる

## GitHub Codespaces で開く

### 手順

- 自分のFork（`github.com/あなたのユーザー名/nagaya-base`）のページを開く
- 緑の「Code」ボタンを押す
- 「Codespaces」タブを選ぶ
- 「Create codespace」を押す

しばらく待つと、ブラウザの中にコード編集画面が開きます。ここが作業場です。ここでAIと一緒にファイルを作り、GitHubに push（保存）します。

## Claude Code などのAIツールで開く

Claude Code のようなAIコーディングツールを使う場合は、自分のFork（`あなたのユーザー名/nagaya-base`）を開く（クローンする）ように指示します。あとは会話でお願いするだけで、AIがファイルの作成・編集・保存を進めます。

## AIに読ませるもの（最初の一手）

どの作業場でも、最初にAIへ次の2つを読ませるのが早道です。

- `docs/gadget-spec.md` — 道具の説明書（この1冊だけで道具が作れるように書かれています）
- `gadgets/_template/` — 道具の雛形（コピーして自分の道具の土台にします）

こう伝えます。「`docs/gadget-spec.md` と `gadgets/_template/` を読んでから、道具を作って」。仕様の要点だけ先に知りたい場合は、案内所の「道具の仕様（やさしい版）」を参照してください。

## 工房の指示文を使う

いちばん確実なのは、長屋の**工房**が用意してくれる**指示文（AIに貼る文章）**を使うことです。自分で文章を考えなくても、正しい手順と制約が入った文章がそのまま手に入ります。

### 手順

- 長屋にログインし、メニュー「工房」を開く
- **新しい道具をつくる**でお題を選び、道具のIDを入れる
- 表示された指示文の「指示文をコピー」を押す
- クラウドの作業場のAIに、その指示文を貼り付けて送る

この指示文には、雛形を複製するコマンド（例: `cp -r gadgets/_template gadgets/あなたのID`）、`manifest.json` の `id` と `name` を書き換える指示、作りたいものの説明、そして守るべき制約（プラットフォームとの通信は gadget-sdk 経由のみ、宣言していない権限は使わない 等）が最初から含まれています。

既にある道具を直したい・複製したいときは、工房の「あなたの道具」から各道具の**改善する／複製して新規**ボタンで、それ専用の指示文が出ます。工房画面の詳細は案内所の「工房の使い方」を参照してください。

## 作ったら必ず push（保存）する

AIと作り込んだら、**GitHubの自分のリポジトリに push（コミット）**します。CodespacesやAIツールのボタン・指示でできます。

**push した内容が、次の「部屋で試運転」に反映されます。** push していない変更は試運転に出てこないので注意してください。動作確認のやり方は案内所の「部屋で試運転（プレビュー）」を参照してください。

## 自分のPCで作りたい場合（任意）

PCに環境をそろえて作りたい人向けに、**セットアップウィザード**があります（工房の「道具をつくる」ブロックの折りたたみ「自分のPC（ローカル）で作りたい場合はこちら」からダウンロード）。clone・必要な部品の用意・起動までを自動で案内します。これは選択肢の一つで、クラウドで作るなら不要です。
