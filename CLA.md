貢献の取り扱いを定めるものです。本プロジェクトに Pull Request を提出する前に、
本契約の全文を読み、同意する必要があります。

**同意方法**: CLA Assistant が Pull Request 上で提示する確認に対し、
GitHubアカウントから電子的に同意してください。この電子的同意は、
本契約への署名として扱われます。

- **許諾者（あなた）**: 本プロジェクトに貢献を提出する個人

  （Honmono協会を含むがこれに限らない）に移転されることがあります。

## 2. 定義

- **「貢献」**とは、あなたが本プロジェクトに提出する、著作物性のある一切の
  成果物（ソースコード、文書、設定、画像、翻訳、その他）をいいます。
- **「提出」**とは、Pull Request、Issue、パッチ、その他の手段により、
  本プロジェクトへの取り込みを意図して送付する行為をいいます。

## 3. 著作権の利用許諾

あなたは、プロジェクト管理者に対し、あなたの貢献について、
**非独占的・世界的・無償・取消不能・サブライセンス可能**な利用許諾を付与します。
この許諾には、複製、翻案、公衆送信、頒布、および**商用利用**（本プロジェクトの
成果物を第三者に有償で提供することを含む）が含まれます。

プロジェクト管理者は、あなたの貢献を含む本プロジェクトを、Business Source
License 1.1 その他プロジェクト管理者が適切と判断するライセンスの下で
頒布することができます。

**あなたの貢献の著作権はあなたに帰属したままです**（本契約は譲渡契約では
ありません）。あなたは自身の貢献を、本プロジェクトの外で自由に利用できます。

あなたは、あなたの貢献に含まれる、あなたが許諾可能な特許について、

必要な範囲で、非独占的・世界的・無償・取消不能の実施許諾を付与します。

あなたは、法令上許される最大限の範囲で、あなたの貢献について、

**著作者人格権を行使しない**ことに同意します
（改変、氏名表示の省略・変更を含みます）。

## 6. あなたの表明・保証

あなたは以下を表明し、保証します。

1. 貢献はあなた自身の創作であるか、または本契約と同等の条件で

3. 貢献が職務上の成果物である場合、または雇用契約・業務委託契約により

あなたは、法律上または本契約上明示された場合を除き、貢献を「現状有姿」で
提供し、商品性・特定目的適合性を含む一切の保証を行いません。

あなたは、プロジェクト管理者が本契約上の地位（受領した許諾を含む）を、

あらかじめ同意します。移転が行われた場合、その旨はリポジトリ上で告知されます。

貢献者は、リポジトリのコミット履歴および CONTRIBUTORS ファイルに記録されます。
本プロジェクトから収益が発生した場合の分配は行いません（クレジット表記を
もって貢献の記録とします）。この方針を変更する場合は、リポジトリ上で
告知の上、以後の貢献に適用します。
